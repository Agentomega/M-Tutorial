==================
TRANSLATE Function
==================

Material prepared from `M Programming Book`_ [WALTERS1997]_

Page 69.


    $TR[ANSLATE]


Example:
GTM>W $TR("hello")
hello
GTM>W $TR("hello","h","H") 
Hello
